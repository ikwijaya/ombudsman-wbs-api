# Ombudsman WBS (Whistleblowing System)

This repository contains the **backend service** for the [Ombudsman Whistleblowing System (WBS)](https://wbs2.ombudsman.go.id/#/).  
It is built using **Node.js**, **Express**, **Sequelize**, and **PostgreSQL**, designed to securely handle whistleblowing reports, feedback, and surveys.
